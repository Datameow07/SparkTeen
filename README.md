# SparkTeen - Career Exploration Platform

![SparkTeen Banner](https://via.placeholder.com/800x200/6366F1/FFFFFF?text=SparkTeen+-+Igniting+Teen+Career+Journeys)

SparkTeen is an interactive career exploration platform designed specifically for teenagers to discover their passions, explore career paths, and develop the skills needed for future success.

## 🌟 Features

- **Career Assessment Quizzes**: Interactive tests to help teens identify their interests and strengths
- **Personalized Career Matches**: AI-powered recommendations based on assessment results
- **Skill Development Paths**: Curated learning resources for various career fields
- **Gamified Learning Experience**: Duolingo-style progress tracking with streaks and achievements
- **Community Engagement**: Connect with peers and mentors in safe, moderated spaces
- **Wellness Integration**: Mindfulness activities to support mental health during career exploration

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/sparkteen.git
cd sparkteen
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🛠️ Built With

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **Authentication**: Firebase Auth
- **Database**: Firebase Firestore
- **Deployment**: Vercel/Netlify

## 🎯 Usage

1. **Sign Up**: Create an account with email or social login
2. **Complete Assessments**: Take interest and skills assessments
3. **Explore Careers**: Browse personalized career recommendations
4. **Track Progress**: Monitor your learning journey with visual progress indicators
5. **Engage with Community**: Join groups and discussions about careers of interest
